Programming Languages

The languages that power our digital world and how they came to be.

From Punch Cards to Python

The first programming languages were closer to machine code than to human language. Ada Lovelace wrote what is considered the first algorithm in the 1840s, intended for Charles Babbage's Analytical Engine.

Since then, thousands of programming languages have been created, each with a unique philosophy about how humans should communicate with machines.

Hello, World!

The tradition of writing a "Hello, World!" program as a first step dates back to Brian Kernighan's 1978 book on C. Here it is in a few languages:

Python

print("Hello, World!")

JavaScript

console.log("Hello, World!");

Rust

fn main() {
    println!("Hello, World!");
}
